## Overview
The **MERN Learning Management System** is a full-stack web application to manage courses, users, and learning progress.  
It allows users to register, log in, browse courses, enroll, and track progress.  

Although built in **React for web**, the architecture and **component-based design are fully transferable to React Native** for mobile applications.

---

## Features

- User Registration and JWT Authentication
- Browse and enroll in courses
- Track course progress in profile
- Responsive UI with React and Tailwind CSS
- RESTful API integration between frontend and backend
- MongoDB database storage
- Modular and scalable project structure

---

## Tech Stack

**Frontend:** React.js, React Router DOM, Axios, Tailwind CSS  
**Backend:** Node.js, Express.js, MongoDB & Mongoose, JWT Authentication  
**Deployment:** Frontend: Netlify | Backend: Render | Database: MongoDB Atlas

## Installation

1. **Clone the repository**
```bash
git clone https://github.com/Sarthakjanrao07/MERN_LMS_APP.git

cd server
npm install

MONGO_URI=<Your MongoDB connection string>
JWT_SECRET=<Your JWT secret key>
PORT=5000

Run the application

Backend:

cd server
npm run dev


Frontend:

cd client
npm start

```
